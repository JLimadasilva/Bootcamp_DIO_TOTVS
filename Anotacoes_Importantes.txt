📌 CONFIGURAÇÃO INICIAL
Comando	Descrição
* git config --global user.name "Seu Nome" <- Define o nome do usuário

* git config --global user.email "seu@email.com" <- Define o e-mail do usuário

* git config --list <- Lista todas as configurações



📁 INICIAR / CLONAR REPOSITÓRIO
Comando	Descrição
* git init <- Inicia um novo repositório Git na pasta atual

* git clone <url> <- Clona um repositório remoto

* git clone <url> --branch <nome> --single-branch <- Clona apenas uma branch específica



📝 TRABALHANDO COM ARQUIVOS
Comando	Descrição
* git status <- Mostra o estado dos arquivos (modificados, adicionados, etc.)

* git add <arquivo> <- Adiciona um arquivo específico à área de staging

* git add . <- Adiciona todos os arquivos modificados à área de staging

* git commit -m "mensagem" <- Cria um commit com a mensagem informada

* git commit --amend -m "nova msg" <- Altera a mensagem do último commit

* git rm <arquivo> <- Remove um arquivo do repositório

* git mv <origem> <destino> <- Move ou renomeia um arquivo



🌿 BRANCHES (RAMIFICAÇÕES)
Comando	Descrição
* git branch <- Lista todas as branches locais

* git branch <nome> <- Cria uma nova branch

* git checkout <branch> <- Alterna para outra branch

* git switch <branch> <- Alterna para outra branch (alternativa moderna)

* git checkout -b <branch> <- Cria e já alterna para a nova branch

* git merge <branch> <- Mescla a branch informada à branch atual

* git branch -d <branch> <- Deleta uma branch



🔄 SINCRONIZAÇÃO COM REMOTO
Comando	Descrição
* git remote -v <- Lista os repositórios remotos configurados

* git remote add origin <url> <- Adiciona um repositório remoto chamado "origin"

* git push origin <branch> <- Envia commits locais para o repositório remoto

* git pull origin <branch> <- Baixa e mescla as alterações do remoto

* git fetch origin <- Baixa as alterações do remoto sem mesclar

* git clone <url> <- Clona um repositório remoto



📜 HISTÓRICO E LOGS
Comando	Descrição
* git log <- Mostra o histórico de commits

* git log --oneline <- Mostra o histórico resumido (1 linha por commit)

* git diff <- Mostra as diferenças entre arquivos modificados

* git diff --staged <- Mostra as diferenças na área de staging

* git show <commit> <- Mostra detalhes de um commit específico



↩️ DESFAZER / CORRIGIR
Comando	Descrição
* git reset <arquivo> <- Remove um arquivo da área de staging

* git reset --soft HEAD~1 <- Desfaz o último commit, mas mantém as alterações

* git reset --hard HEAD~1 <- Desfaz o último commit e descarta as alterações

* git checkout -- <arquivo> <- Descarta alterações não commitadas de um arquivo

* git restore <arquivo> <- Restaura arquivo para o último commit (moderno)

* git restore --staged <arquivo> <- Remove arquivo da área de staging (moderno)
